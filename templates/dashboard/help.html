Help & Support

What is BomRadar?

BomRadar is an AI-powered fraud detection system designed to monitor financial transactions and flag suspicious activity in real time.

How does fraud detection work?

Transactions are analyzed using a machine learning model trained on historical credit card data. If a transaction appears risky, it is flagged for review.

What happens if my transaction is flagged?

The transaction will be temporarily blocked and you will receive a notification explaining the reason.
